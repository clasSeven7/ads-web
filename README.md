# Projeto ADS-Web

Este projeto foi desenvolvido para o curso de Análise e Desenvolvimento de Sistemas (ADS) da Faculdade Unifip.

## Sobre o Projeto

O objetivo deste projeto é criar um site para divulgar informações e documentos relacionados ao curso de ADS. O site incluirá dados sobre matrículas, disciplinas, monitoria, extensão e outros aspectos importantes do curso.

## Coleta de Informações

Para coletar as informações necessárias, é essencial entrar em contato com Sormany. Serão necessárias várias conversas para garantir que todos os documentos e dados relevantes sejam obtidos e divulgados corretamente no site. É importante respeitar as demandas de Sormany durante este processo.

## Como Contribuir

1. **Planejamento**: Organize-se e planeje como e quando entrar em contato com Sormany.
2. **Coleta de Dados**: Reúna todas as informações e documentos necessários sobre matrículas, disciplinas, monitoria, extensão, etc.
3. **Desenvolvimento**: Utilize as informações coletadas para desenvolver e atualizar o site do curso.

## Contato

Para mais informações, entre em contato com a equipe do projeto ou com Sormany diretamente.
