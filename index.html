Análise e Desenvolvimento de Sistemas ADS

O curso de Análise e Desenvolvimento de Sistemas é uma graduação tecnológica que forma profissionais para atuar no desenvolvimento de sistemas de informação.

Acesse aqui

Objetivo Curso

O curso de Análise e Desenvolvimento de Sistemas tem como objetivo formar profissionais capazes de analisar, projetar, desenvolver, implementar e manter sistemas de informação, com conhecimentos de programação, banco de dados, redes, engenharia de software, sistemas operacionais, entre outros.

Perfil Profissional

O profissional formado em Análise e Desenvolvimento de Sistemas deve possuir as seguintes competências:

  • Capacidade de análise e síntese
  • Raciocínio lógico
  • Capacidade de trabalhar em equipe
  • Conhecimento de língua estrangeira
  • Capacidade de comunicação oral e escrita
  • Capacidade de aprendizado contínuo
  • Capacidade de liderança
  • Conhecimento de ética e cidadania

Áreas Atuação

O profissional formado em Análise e Desenvolvimento de Sistemas pode atuar em diversas áreas, tais como:

Desenvolvimento de sistemas Desenvolvimento de aplicativos móveis Desenvolvimento de jogos Desenvolvimento de sistemas web Administração de banco de dados Administração de redes de computadores

Pré-requisitos

Para ingressar no curso de Análise e Desenvolvimento de Sistemas, é necessário ter concluído o ensino médio.

Duração curso

O curso de Análise e Desenvolvimento de Sistemas tem duração de 2 anos (4 semestres).

Titulação

Ao concluir o curso de Análise e Desenvolvimento de Sistemas, o aluno recebe o diploma de Técnico em Análise e Desenvolvimento de Sistemas.

Grade curricular

A grade curricular do curso de Análise e Desenvolvimento de Sistemas é composta pelas seguintes disciplinas:

  • Algoritmos e Lógica de Programação
  • Programação Orientada a Objetos (Java)
  • Banco de Dados I
  • Engenharia de Software
  • Programação Web I
  • Desenvolvimento Mobile
  • Desenvolvimento de Jogos
  • Projeto Integrador I
  • Projeto Integrador II
  • Projeto Integrador III
  • Projeto Integrador IV
