Informações sobre a Disciplina

Esta disciplina tem como objetivo introduzir os conceitos básicos de algoritmos e lógica de programação, fundamentais para o desenvolvimento de software. Os tópicos abordados incluem:

  • Introdução à lógica de programação
  • Estruturas de controle
  • Estruturas de dados
  • Algoritmos de ordenação e busca
  • Introdução à programação orientada a objetos

Materiais

  • Apostila de Algoritmos
  • Slides das Aulas
  • Exercícios Resolvidos

  • Projeto 2: Jogo da Velha
  • Projeto 3: Sistema de Cadastro
